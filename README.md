## Installation

```bash
$ npm i
$ npm run build
```

_Chrome_ → _Extensions_ → Check _Developer mode_ → _Load unpacked extension_ → Select the `dist` directory.

### Development

```bash
$ npm run dev
```

After every change, the extension needs to be updated in chrome://extensions/
