### Installation

```bash
$ npm i
$ npm run build
```

*Chrome* → *Extensions* → Check *Developer mode* → *Load unpacked extension* → Select the `dist` directory.

Once installed, the extension can be reloaded by clicking *Update extensions now* (⌘R).

### Development

```bash
$ npm run watch
```
